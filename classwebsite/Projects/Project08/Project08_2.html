Project 8 Challenge 2:

The table below was created by pasting text directly from the PDF instructions into a paragraph.  Next, it was converted into a table from the <Table><Convert> menu option.  For this to work properly, CRLFs had to be inserted at the appropriate end of lines and then used by the feature as the delimiter defining the table. 

Once the table was created, a second column was added by selecting the table, opening the Table Properties Dialog Box and increasing the Columns from 1 to 2.  The cells of the new column were merged and a nested table pasted in.
