Quick Setup

shop - Shop files for servers with PHP register_globals = On (Try these files first)
shop_reg_global_of - Shop files for servers with PHP register_globals = Off

Modify the file config.php in the folder shop or shop_reg_global_of with text editor such as WordPad or NotePad.

$host = "192.168.0.50";   //mysql database host
$user = "root";   //mysql database username
$pass = "Ghy6Rgbv";   // mysqldatabase password
$database = "midicart_demo";   //mysql database
$receipt = "demo@midicart.com";   //e-mail account for receiving orders
$txt_currency="USD";   //currency.
$upload_path = "/home/username/html/shop/images/";   // upload path to the images directory
$add_tax = "0";   //tax in %. Enter 16.00 for 16%. If not used = 0
$add_freight = "0";   //firm freight charge in format 5.00. If not used = 0
Upload the folder shop or shop_reg_global_of to your website
The images directory needs to be chmoded to 0777. (write enabled for all users)
Point your browser to the new folder in your website. (http://www.domain.com/shop/)
Click on "Click here to install MidiCart PHP tables in MySQL" in the main frame to install MidiCart PHP tables in MySQL
