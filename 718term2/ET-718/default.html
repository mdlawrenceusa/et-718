ET-718 Database Technology - PHP & MySQL

Course Objectives:  Students will learn the fundamentals of modern database technology. They will study and create solutions using PHP and MySQL programming. Topics covered will include PHP, Arrays, Handling files, MySQL, Product Catalog, Shopping carts. Authentication and PHP Graphics.

Prerequisite:  ET-710 Web Technology: Building and Maintaining Web Sites

Text/Materials: Text: ET-718 Database Technology, Laboratory Guide Version 1, By Robert M. Kueper
Software: Server PHP, MySQL, Workstation: Microsoft Expression Web

Teaching Methods: Lectures: Important material from the text and outside sources will be covered in class. Students should plan to take careful notes as not all material can be found in the texts or readings. Discussion is encouraged as is student-procured, outside material relevant to topics being covered. Blackboard will have a discussion area for all students to communicate with each other online.
Demonstrations: Much of the class time will be used to demonstrate the practical use of the software for this course. Students are encouraged to carefully observe the demonstrations and also to go through the tutorial material in the textbook.

Assignments: Labs and readings will be assigned to help support and supplement material found in the text.
Quizzes: Quizzes will be given to help ensure students stay up with assigned material and reinforce learning.

  12. Introduction to the MIDI E-Commerce site

  13. Install the MIDI E-commerce Database and Tables

  14. Configure and test the MIDI E-Commerce Site

  15. Final Exam
